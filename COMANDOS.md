


## Zabbix
one-liner

```bash
curl -fsSL https://raw.githubusercontent.com/runtechx/OpenFirst/master/scripts/zabbix_al10.sh -o zabbix.sh && bash zabbix.sh
```
Log (abrir em outro terminar)
```bash
tail -f /var/log/deploy-zabbix.log
```
