
# Objectivo - 1 

PT

## Compreender e utilizar ferramentas essenciais

- Aceder a um terminal shell e executar comandos com a sintaxe correta
- Utilizar redirecionamento de entrada e saída (>, >>, |, 2>, etc.)
- Utilizar grep e expressões regulares para analisar texto
- Aceder a sistemas remotos utilizando SSH
- Iniciar sessão e alternar entre utilizadores em ambientes multiutilizador
- Arquivar, comprimir, extrair e descomprimir ficheiros utilizando tar, gzip e bzip2
- Criar e editar ficheiros de texto
- Criar, eliminar, copiar e mover ficheiros e diretórios
- Criar ligações físicas (hard links) e simbólicas (soft links)
- Listar, definir e alterar permissões padrão ugo/rwx
- Localizar, ler e utilizar documentação do sistema, incluindo man, info e ficheiros em /usr/share/doc

EN

## Understand and use essential tools

- Access a shell prompt and issue commands with correct syntax
- Use input-output redirection (>, >>, |, 2>, etc.)
- Use grep and regular expressions to analyze text
- Access remote systems using SSH
- Log in and switch users in multiuser targets
- Archive, compress, unpack, and uncompress files using tar, gzip, and bzip2
- Create and edit text files
- Create, delete, copy, and move files and directories
- Create hard and soft links
- List, set, and change standard ugo/rwx permissions
- Locate, read, and use system documentation including man, info, and files in /usr/share/doc
