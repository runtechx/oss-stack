# INFO 

Para verificar mais informações sobre um IP encontrado na blocklist da RT, utilize o site [AbuseIPDB](https://www.abuseipdb.com/?utm_source=chatgpt.com).

Neste site é possível consultar indicadores como:

* Quantidade de denúncias associadas ao IP
* Percentagem de confiança de abuso (*Confidence of Abuse*)
* ISP/Operadora responsável
* Tipo de utilização do IP (Datacenter, Hosting, Residencial, etc.)
* ASN
* Domínio associado
* País e cidade de origem

Exemplo:

O IP `101.126.138.178` foi reportado 82 vezes e possui uma confiança de abuso de 75%, estando associado ao ISP `Beijing Volcano Engine Technology Co., Ltd.`, infraestrutura `Data Center/Web Hosting`, domínio `bytedance.com` e origem em Pequim, China.

> Vá a `/guias` para encontrar guias sobre como tirar melhor partido da plataforma.
